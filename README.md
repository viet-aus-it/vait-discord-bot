# VAIT Discord Bot

A community-driven Discord bot for the [Vietnamese Australians in IT](https://discord.gg/vait) Discord server.

## Tech Stack

- [TypeScript](https://www.typescriptlang.org/), [Node.js](https://nodejs.org/) 22+
- [discord.js](https://discord.js.org/) v14
- [Prisma](https://www.prisma.io/) ORM with [PostgreSQL](https://www.postgresql.org/)
- [Vitest](https://vitest.dev/) for testing

## Quick Start

> Prerequisites: [Node.js](https://nodejs.org/) 22+, [PNPM](https://pnpm.io/) 10+, [Docker](https://www.docker.com/) 20+
