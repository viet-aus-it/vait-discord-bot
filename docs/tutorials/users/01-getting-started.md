# Getting Started

Welcome to the VAIT Discord Bot! This guide walks you through your first interactions with the bot.

## What is the VAIT Bot?

The VAIT Bot is a community bot for the [Vietnamese Australians in IT](https://discord.gg/vait) Discord server. It provides fun commands, utilities, and moderation tools.

## How Slash Commands Work

Discord slash commands start with `/`. When you type `/` in a chat, Discord shows you a list of available commands with descriptions.

1. Type `/` in any text channel
2. Browse or search for a command
3. Fill in any required options
4. Press Enter to run it

## Try Your First Commands

### Ask the Magic 8ball

```
/8ball question: Will it rain today?
```

The bot replies with a random answer like "Yes", "Hell No", or "Go for it".

### Check the Weather

```
/weather location: Melbourne
```

Displays current weather conditions for the specified location.

### Get a Quote of the Day

```
/qotd
```

Shows an inspirational quote to brighten your day.
